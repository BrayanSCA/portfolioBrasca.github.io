## Description
***
Este es mi portafolio profesional como **Tecnólogo en Análisis y Desarrollo de Sistemas**. Aquí presento una curaduría de mis trabajos, divididos en mini-proyectos de lógica y landing pages enfocadas en UI/UX. Mi objetivo es demostrar una base sólida en JavaScript y una metodología de trabajo organizada y escalable.

## Architecture
***
El proyecto sigue una estructura modular para facilitar el mantenimiento:
- `base/`: Variables globales y reset.
- `layout/`: Estructura principal (Header, Hero, Footer).
- `components`: Elementos reutilizables como botones y tarjetas.

## Technologies used
***
* **Diseño Responsivo:** Adaptado para móviles, tablets y escritorio.
* **Sección Hero:** Presentación personal de alto impacto.
* **Zig-Zag Layout:** Presentación dinámica de proyectos Front-end.
* **Sticky Navigation:** Menú siempre accesible para el usuario.
